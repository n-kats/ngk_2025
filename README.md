# NGK2025S ソースコード・データ

## 実行方法
1. OpenAIのAPIキーを取得し、環境変数`OPENAI_API_KEY`にセットします。
1. poetryをインストールします。
1. `git clone --recursive git@github.com:n-kats/ngk_2025.git`
1. `poetry install`を実行します。
1. `poetry run python main.py`を実行します。
